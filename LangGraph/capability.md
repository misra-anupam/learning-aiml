    - Order status / tracking

    - Partial order / item missing from shipment

    - Exchange request (size/color/variant)

    - Return window expired query

# 4. Account & access
    - Login/OTP issues
    - Password reset
    - Update profile/email/phone
    - Account deletion request
    - Merge duplicate accounts

# 5. Product & pre-purchase
    - Product availability / restock query

    - Price match / price drop query

# 6. Policy & informational
    - Policy clarification (returns, warranty, shipping)
    - Warranty claim / registration
    - Terms of service / data privacy query

# 7. Loyalty & subscriptions
    - Loyalty points balance / redemption
    - Subscription pause/cancel/modify
    - Membership renewal issue

# 8. Complaints & escalations
    - Service quality complaint (delivery partner, packaging)
    - Complaint against support agent
    - Request to speak with human / escalate
    - Repeat unresolved issue follow-up

# 9. Fraud & security
    - Unauthorized transaction report
    - Account compromise / phishing report
    - Counterfeit product report

# 10. Ticket & meta
    - Ticket status tracking
    - Reopen closed ticket
    - General feedback / NPS response
